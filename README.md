# MangaStyle

Ecommerce de playeras de anime con una direccion visual editorial y enfoque streetwear. El proyecto esta construido con Next.js y mantiene animaciones, lookbook, carrito y paginas de marca.

## Stack

- Next.js 16
- React 19
- GSAP
- Zustand

## Desarrollo

```bash
npm run dev
```

Abre `http://localhost:3000` para ver la tienda.

## Secciones principales

- `/` home con hero, destacados y narrativa de marca
- `/wardrobe` catalogo completo de playeras
- `/unit` detalle de producto destacado
- `/genesis` pagina editorial sobre la marca
- `/lookbook` experiencia visual del drop
- `/touchpoint` contacto y comunidad

## Objetivo del proyecto

Reposicionar la plantilla original de moda moderna como una tienda llamada `MangaStyle`, enfocada en vender playeras de anime con catalogo curado, copy en espanol y branding consistente.
